Several factors determine the execution environment of an application.
<ul>
  <li>
    Compiler (version, flags, <em>etc</em>)
  </li>
  <li>
    CPU architecture
  </li>
  <li>
    Application dependencies (and <em>their</em> versions)
  </li>
  <li>
    Middleware
  </li>
</ul>

The execution environment should be made explicit
